Detailed Description

The City-Selection Travelling Salesman Problem.

This is a class representing a new variant of the TSP which was firstly introduced (by us) in the context of asteroid/space debris selection problems.

Given a fully connected graph (V,E), where E is a list of weighted edges and V a list of valued vertices, the problem is that of finding a path of length accumulating the largest possible vertex value. We encode a solution to this problem as an Hamiltonian path and evaluate its merit finding, along it, the best sequence satisfying the constraint.
